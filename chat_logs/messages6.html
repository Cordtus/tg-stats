The process for generating a CosmWasm wallet address is as follows:

1. Generate a private key from a mnemonic phrase.
2. Decode the private key using the Secp256k1 standard's `SecretKey` method.
3. Compute the public key using the Secp256k1 standard's `PublicKey` method.

---These steps comply with the Secp256k1 standard---

4. Hash the public key using SHA-256 and RIPEMD-160, then encode it using bech32 to obtain the address.

### Reason for Invalid Signature

The invalid signature issue arises because CosmWasm wallet addresses use SHA-256 and RIPEMD-160 hashing, making it impossible to recover the correct public key using Secp256k1 standard methods alone.

Since SHA-256 and RIPEMD-160 are not part of the Secp256k1 standard, they lead to inconsistencies in the signing and recovering process.
